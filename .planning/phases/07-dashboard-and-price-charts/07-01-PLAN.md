**PHP Class Section:**

```php
<?php
declare(strict_types=1);

use App\Models\WatchedItem;
use Illuminate\Support\Collection;
use Livewire\Attributes\Computed;
use Livewire\Attributes\Layout;
use Livewire\Volt\Component;

new #[Layout('layouts.app')] class extends Component
{
    public ?int $selectedItemId = null;
    public string $timeframe = '7d';

    // ... methods below
};
```

**Computed property — watchedItems():**
- Query: `auth()->user()->watchedItems()` — NEVER use `WatchedItem::query()` (DASH-06 isolation).
- Eager load: `->with(['priceSnapshots' => fn ($q) => $q->latest('polled_at')->limit(2)])` — only 2 snapshots for trend computation (avoids N+1).
- Order by name.

**Method — selectItem(int $id):**
- Toggle: if `$this->selectedItemId === $id`, set to null (close chart); otherwise set to $id and call `loadChart()`.

**Method — setTimeframe(string $frame):**
- Set `$this->timeframe = $frame` and call `loadChart()`.

**Method — loadChart() (private):**
- Compute cutoff using match: `'24h' => now()->subHours(24)`, `'30d' => now()->subDays(30)`, `default => now()->subDays(7)`.
- Query: `auth()->user()->watchedItems()->findOrFail($this->selectedItemId)->priceSnapshots()->where('polled_at', '>=', $cutoff)->orderBy('polled_at')->get(['polled_at', 'median_price', 'min_price'])`.
- Dispatch browser event: `$this->dispatch('chart-data-updated', median: [...], min: [...])`.
- Map snapshots to arrays of `['x' => $s->polled_at->timestamp * 1000, 'y' => $s->median_price]` (millisecond timestamps for ApexCharts datetime x-axis). Use `->timestamp * 1000` (safe fallback) rather than `getPreciseTimestamp(3)` which may not be available.

**Method — formatGold(int $copper): string:**
- `$g = intdiv($copper, 10000);`
- `$s = intdiv($copper % 10000, 100);`
- `$c = $copper % 100;`
- Build parts array: if $g > 0, add `number_format($g) . 'g'`; if $s > 0, add `$s . 's'`; if $c > 0 or parts empty, add `$c . 'c'`.
- Return `implode(' ', $parts)`.

**Method — trendDirection(WatchedItem $item): string:**
- If item has < 2 snapshots, return `'flat'`.
- Compare `$snapshots->first()->median_price` (latest, due to `latest('polled_at')` ordering) to `$snapshots->last()->median_price` (previous).
- Return `'up'`, `'down'`, or `'flat'`.

**Method — trendPercent(WatchedItem $item): ?float:**
- If item has < 2 snapshots, return null.
- Compute: `round((($current - $previous) / $previous) * 100, 1)`.
- Return null if $previous === 0 (avoid division by zero).

**Method — dataFreshness(): string:**
- Query the latest `polled_at` across user's watched items' snapshots.
- Use `Carbon::parse($latest)->diffForHumans()` if exists, otherwise `'Never'`.

**Script Block Section:**

Bare `<script>` block (NOT wrapped in `@script`/@endscript` — Volt SFCs use bare tags):

```javascript
let chart = null;

function formatGoldJs(copper) {
    if (copper === null || copper === undefined) return '\u2014';
    const g = Math.floor(copper / 10000);
    const s = Math.floor((copper % 10000) / 100);
    const c = copper % 100;
    const parts = [];
    if (g > 0) parts.push(g.toLocaleString() + 'g');
    if (s > 0) parts.push(s + 's');
    if (c > 0 || parts.length === 0) parts.push(c + 'c');
    return parts.join(' ');
}

$wire.$on('chart-data-updated', ({ median, min }) => {
    const options = {
        series: [
            { name: 'Median', data: median },
            { name: 'Min',    data: min },
        ],
        chart: {
            type: 'line',
            height: 300,
            background: '#1a1a2e',
            toolbar: { show: false },
            animations: { enabled: true },
        },
        theme: { mode: 'dark' },
        colors: ['#f7a325', '#60a5fa'],  // wow-gold, blue-400
        stroke: { curve: 'smooth', width: 2 },
        markers: { size: 0 },
        xaxis: {
            type: 'datetime',
            labels: {
                style: { colors: '#9ca3af' },
                datetimeUTC: false,
            },
        },
        yaxis: {
            labels: {
                style: { colors: '#9ca3af' },
                formatter: (val) => Math.floor(val / 10000).toLocaleString() + 'g',
            },
        },
        tooltip: {
            theme: 'dark',
            custom: ({ series, seriesIndex, dataPointIndex, w }) => {
                const medianVal = series[0] ? series[0][dataPointIndex] : null;
                const minVal = series[1] ? series[1][dataPointIndex] : null;
                const ts = w.globals.seriesX[0][dataPointIndex];
                const date = new Date(ts);
                const timeStr = date.toLocaleDateString() + ' ' + date.toLocaleTimeString([], { hour: '2-digit', minute: '2-digit' });
                return '<div class="px-3 py-2 text-sm">'
                    + '<div class="text-gray-400 mb-1">' + timeStr + '</div>'
                    + '<div><strong>Median:</strong> ' + formatGoldJs(medianVal) + '</div>'
                    + '<div><strong>Min:</strong> ' + formatGoldJs(minVal) + '</div>'
                    + '</div>';
            },
        },
        grid: { borderColor: '#374151' },
    };

    const el = document.querySelector('#price-chart');
    if (!el) return;

    if (chart === null) {
        chart = new ApexCharts(el, options);
        chart.render();
    } else {
        chart.updateOptions(options);
    }
});
```

Key ApexCharts notes:
- ALWAYS check `if (chart === null)` — use `updateOptions()` for subsequent changes, never re-instantiate (causes flicker).
- Y-axis formatter shows gold-only abbreviation (e.g., "1,453g") — full breakdown in tooltip.
- Tooltip uses custom formatter with gold/silver/copper breakdown.
- `datetimeUTC: false` so x-axis shows local time.
