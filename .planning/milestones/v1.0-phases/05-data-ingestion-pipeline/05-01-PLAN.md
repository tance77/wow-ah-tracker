<task type="auto" tdd="true">
  <name>Task 1: Implement PriceAggregateAction with frequency-distribution median</name>
  <files>app/Actions/PriceAggregateAction.php</files>
  <behavior>
    - Empty listings array returns {min_price: 0, avg_price: 0, median_price: 0, total_volume: 0}
    - Single listing returns that listing's price as min, avg, and median
    - Multiple listings: min_price is the lowest unit_price across all listings
    - Multiple listings: avg_price is (int) round(sum(price * qty) / sum(qty)) — weighted by quantity, cast to int
    - Multiple listings: median_price uses cumulative quantity traversal (sort by unit_price asc, accumulate qty, return price where cumulative >= ceil(totalVolume/2))
    - Multiple listings: total_volume is sum of all quantities
    - Large-quantity bucket dominates median: [{price:100, qty:500}, {price:200, qty:10}, {price:300, qty:5}] -> median=100 (not 200)
    - All return values are integers, never floats
  </behavior>
  <action>
    Create `app/Actions/PriceAggregateAction.php` as an invokable action class:

    1. Namespace: `App\Actions`, `declare(strict_types=1)`
    2. Method signature: `public function __invoke(array $listings): array`
       - Input: `array<int, array{unit_price: int, quantity: int}>` — already-filtered listings for ONE item
       - Output: `array{min_price: int, avg_price: int, median_price: int, total_volume: int}`
    3. Handle empty listings: return all-zero array
    4. Single-pass loop to compute totalVolume, totalValue, minPrice:
       ```php
       $totalVolume = 0;
       $totalValue = 0;
       $minPrice = PHP_INT_MAX;
       foreach ($listings as $listing) {
           $price = $listing['unit_price'];
           $quantity = $listing['quantity'];
           $totalVolume += $quantity;
           $totalValue += $price * $quantity;
           if ($price < $minPrice) {
               $minPrice = $price;
           }
       }
       ```
    5. Average: `(int) round($totalValue / $totalVolume)` — round first, then cast to int (not truncate)
    6. Median via private `computeMedian()` method:
       - `usort($listings, fn(array $a, array $b): int => $a['unit_price'] <=> $b['unit_price'])`
       - `$medianPosition = (int) ceil($totalVolume / 2)`
       - Cumulative traversal: accumulate quantity, return unit_price of first bucket where cumulative >= medianPosition
       - Fallback: return last listing's unit_price (should not be reached with non-empty input)

    CRITICAL: Do NOT use a naive `sort($prices)` approach. The median MUST account for quantity. A listing with qty=500 at price 100 means 500 units at that price — it dominates over 10 units at price 200.

    Follow existing project conventions:
    - `declare(strict_types=1)` at top of file
    - PHPDoc `@param` and `@return` type hints on `__invoke`
    - No database queries inside this class — it is pure math
  </action>

  </verify>

    6. handle() flow:
       a. Query ALL WatchedItem rows across all users: `$watchedItems = WatchedItem::all()`
       b. Early return with Log::info if empty
       c. Collect unique blizzard_item_ids: `$itemIds = $watchedItems->pluck('blizzard_item_id')->unique()->values()->all()`

       f. Group listings by item ID:
          ```php
          $grouped = [];
          foreach ($listings as $listing) {
              $id = $listing['item']['id'];
              $grouped[$id][] = ['unit_price' => $listing['unit_price'], 'quantity' => $listing['quantity']];
          }
          ```
       g. Capture `$polledAt = now()` ONCE (all snapshots in one run share same timestamp)
       h. Iterate over ALL `$watchedItems` (not unique IDs) and write one PriceSnapshot per WatchedItem:
          ```php
          foreach ($watchedItems as $watchedItem) {
              $itemListings = $grouped[$watchedItem->blizzard_item_id] ?? [];
              $metrics = ($aggregateAction)($itemListings);
              PriceSnapshot::create([
                  'watched_item_id' => $watchedItem->id,
                  'polled_at' => $polledAt,
                  ...$metrics,
              ]);
          }
          ```
